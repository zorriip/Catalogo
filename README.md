Este sistema web basado en html php y mysql muestra un catálogo de macetas de bonsai.Se muestran fotos, codigo y descrpcion de las macetas registradas. al clic en la foto se abre una pagina de detalles y mas fotos de la maceta.
tambien cuenta cona seccion de Mantenimiento accesible al administrador para publicar o modificar los registros.
